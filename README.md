# Predicting Polarity in Political News Through Machine Learning Models Logistic Regression and BERT

#### This repository contains the code to create a logistic regression and BERT model to evaluate the sentences in the dataset provided [here](https://huggingface.co/datasets/fhamborg/news_sentiment_newsmtsc)

#### Steps to download and process the data is in final_code.py

#### Data provided by the NewsMTSC benchmark (Hamborg et al., 2021).

#### Hamborg, F., & Donnay, K. (2021, April). NewsMTSC: (Multi-)target-dependent sentiment classification in news articles. In Proceedings of the 16th Conference of the European Chapter of the Association for Computational Linguistics (EACL 2021). Virtual Event.
